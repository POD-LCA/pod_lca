**********
Project
**********

**Project** creates a new project containing the different :class:`Model <model.model.Model>`'s, :class:`Calculator <calculator.calculator.Calculator>`, :class:`Database <databaseManager.databaseManager.DatabaseManager>` and :class:`Visualiser`.

The line of code below creates a new **Project**.

.. code-block:: python

    from material.projectManager.projectManager import Project

    project = Project()
