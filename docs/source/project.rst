**********
Project
**********

**Project** creates a new project containing the different :class:`Model <material.model.Model>`'s, :class:`Calculator <material.calculator.Calculator>`, :class:`Database <material.databaseManager.DatabaseManager>` and :class:`Visualiser`.

The line of code below creates a new **Project**.

.. code-block:: python

    from material.projectManager.projectManager import Project

    project = Project()
